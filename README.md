# Plaice

Print Once, Reconfigure As Your Space Evolves

# Description
Plaice is a 3d-printable modular construction system for living space organization. Unlike other similar
systems, Plaice enables radical free-form constructions out of basic structural elements called _plates_,
which naturally slide together to allow for large assemblies to be built from smaller parts. Using just a few standard Plaice components,
it's possible to make:
- Stacking Boxes
- Multi-tiered shelves
- Cube Organizers
- Drawers
- Spice Racks
- Hinged Clamshell Boxes
- Drawer Dividers
... among many, many more things!

Open, endlessly remixable, with seamless compatibility with other systems like Gridfinity and OpenGrid, Plaice makes
your organization dreams achievable by allowing experimentation without waste through modular composition. 
Designed with simplicity and internal coherence in mind, Plaice stands out from similar "universal" organization
systems in its relatively small number of core components with thoughtful, carefully-engineered designs.

Printing for most parts assumes settings for a .4mm nozzle, two perimeters, and a layer height of .1mm. Most components will print without the need for slicer-generated supports, but check individual component pages for details.
