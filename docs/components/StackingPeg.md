# Stacking Pegs

<img src="/docs/images/stackingpegs/stackingpegbottom.jpg" alt="A Stacking Peg, Bottom View" width="256" height="256"> <img src="/docs/images/stackingpegs/stackingpegside.jpg" alt="A Stacking Peg, Side View" width="256" height="256"> 

A stacking peg is effectively a stub version of a Plaice nubbed connector with smoothed geometry which makes it
more hand-friendly while also retaining the geometry needed to seat into a corner stacking cap.
